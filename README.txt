1. 프로젝트 설명

본 프로젝트는 Spring Boot를 사용한 백엔드 서버 구동 애플리케이션이다.
본 프로젝트는 Gradle을 활용한 Spring Boot 프로젝트이다.


본 프로젝트 코드는 다음과 같은 기능을 포함하고 있다.

- 프론트단에서 테스트 시작을 받을 시 자동으로 DB에 정보 추가
- 프론트단에서 받은 사진을 AI 서버로 전송한 후, 분석 결과를 DB에 저장
- 프론트단에서 받은 피드백 데이터를 확률값으로 변환 후, DB에 저장
- 결과 출력 요청 시 DB로부터 데이터를 찾아서 전송



2. 프로젝트 사용법

본 프로젝트를 실행하기 앞서, 다음 프로그램이 필요하다.

- H2 Database (다른 데이터베이스 사용 시 약간의 수정 필요)
- AI 모델 서버 구동 파일 및 모델
- 프론트엔드 서버 구동 파일


본 프로젝트를 구동하기 전에 application.yml 파일의 일부를 수정해야 한다.

- spring.datasource.url에 사용할 DB에 대한 설정
- jpa.hibernate.ddl-auto를 최초 구동/초기화 시 create로 설정, 이외의 상황에는 none으로 설정
- file.upload-dir를 데이터를 저장할 경로로 설정 (자세한 내용은 코드 확인)
- server-link.model을 통신할 PyTorch 서버의 url로 설정


본 프로젝트를 구동하기 전에 DB 서버를 미리 실행해야 한다.
